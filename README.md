```txt

├── components/
│   ├── ui/         # Reusable UI components
│   └── features/   # Feature-specific components
├── routes/         # Svelte pages (file-based routing)
├── lib/            # Utility functions and helpers
├── static/
│   └── assets/     # Static assets like images, fonts
```
